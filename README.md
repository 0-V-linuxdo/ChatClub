# ChatClub

ChatClub 是一个多 AI 聊天工作台浏览器扩展。

一次输入，多平台发送；集中对比回答、总结内容、优化提示词。

## 功能

- 多平台同屏聊天
- 统一输入与发送
- 提示词优化
- 聊天内容总结
- 提示词库
- 自定义平台配置
- 配置导入 / 导出
- 顶部栏自定义

## 安装

1. 下载或克隆本仓库。
2. 打开浏览器扩展管理页面。
3. 启用开发者模式。
4. 选择“加载已解压的扩展程序”。
5. 选择本项目文件夹。

## 目录

- `app/`：主界面
- `background/`：扩展后台
- `content/`：内容脚本
- `shared/`：共享逻辑
- `styles/`：样式
- `userscripts/`：内置总结脚本

## 开发

无需构建。修改后，在扩展管理页面重新加载即可。
